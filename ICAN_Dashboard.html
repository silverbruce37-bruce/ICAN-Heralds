# 아이캔헤럴즈 (ICAN Heralds) 디자인 명세서

## 1. 개요
필리핀 거주 한국인 및 관광객을 대상으로 한 '타임즈(The Times)' 스타일의 뉴스 플랫폼. 오늘자 필리핀 주요 뉴스를 영문과 한글로 제공하며, 헤드라인 중심의 강력한 시각적 임팩트를 제공한다.

## 2. 핵심 기능
- **헤드라인 집중형 레이아웃:** 오늘 가장 중요한 뉴스를 페이지 상단에 대형 이미지와 함께 배치.
- **영/한 바이링구얼 토글:** 버튼 하나로 전체 기사의 언어를 영문 또는 한국어로 전환. (기사 데이터는 매핑되어 관리)
- **타임즈 스타일 UI:** Playfair Display 서체, 더블 보더 헤더, 종이 질감 배경 등을 통해 고전 신문의 신뢰감 부여.
- **필리핀 현지 정보:** 용맹의 날(Day of Valor), 에너지 비상사태, 한국인 관광 경찰 등 실시간 데이터 반영.

## 3. 기술 스택 및 구조
- **Frontend:** Vanilla HTML5, CSS3 (Flexbox/Grid), JavaScript (ES6+).
- **디자인 시스템:**
  - 폰트: 'Playfair Display' (제목용), 'Noto Sans KR' (한글 본문용), 'Georgia' (영문 본문용).
  - 색상: #1a1a1a (텍스트), #fdfbf7 (배경), #0047ab (강조색).
- **데이터 구조:** 기사 제목, 요약, 본문을 영문(en)과 한국어(kr) 키를 가진 JSON 객체 형태로 관리하여 토글 시 실시간 교체.

## 4. 상세 레이아웃 (헤드라인 집중형)
1. **Header:** 로고(ICAN HERALDS), 날짜, 영/한 토글 버튼.
2. **Main Article:** 
   - 초대형 헤드라인 + 부제목.
   - 메인 히어로 이미지 (캡션 포함).
   - 2단 구성의 본문 (한글/영문 선택적 노출).
3. **Secondary News (Grid):** 경제, 문화, 안전 분야의 보조 기사 3개 배치.

## 5. 성공 기준
- 영/한 토글 버튼 클릭 시 0.1초 이내에 언어 전환 완료.
- 모바일 및 데스크톱에서 '신문'과 같은 가독성 유지.
- 필리핀 현지 소식이 정확하게 반영됨.
